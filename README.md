# One Page Grayscale

## Credits

[Grayscale](http://startbootstrap.com/template-overviews/grayscale/) is a multipurpose, one page HTML theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/). This template features various content sections and a Google Maps section with a custom map marker.

##Informations générales

Ce thème permet de présenter, sur une seule page, une liste de billets sans fioritures pour un petit site évènementiel par exemple. N'étant pas destiné à un blog, une grande partie des fonctionnalités de Dotclear sont inutilisées :

    pas de catégories, de tags, ni de notion de date ou d'auteur
    pas de commentaires ni de rétroliens
    pas de widgets de présentation

Vous pouvez en revanche ajouter des pièces jointes à vos billets, y insérer toutes sortes de contenus, et afficher autant de billets que vous le désirez. La limitation est celle définie dans les paramètres du blog pour le nombre de billets à afficher sur la page d'accueil.
Configuration du menu

Le plugin Simple Menu est utilisé pour afficher le menu horizontal en haut de page. Néanmoins, en raison de la spécificité du thème, seule l'option Saisie libre est utilisable.

Les billets sont identifiés par leur ordre d'affichage. Pour ajouter un lien vers une section de la page (un billet), créez des éléments de menu avec un lien vers #post-x où x est le numéro d'ordre du billet dans la liste (voir le mini-site de démonstration pour plus de détails)
